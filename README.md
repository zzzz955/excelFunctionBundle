# excelFunctionBundle
엑셀 업무 자동화 및 무결성 기능

# mainwindow.py
메인 GUI

# dialogs.py
팝업 다이얼로그

# tabs.py
기능별로 분류 한 탭 노출
 * 데이터 병합본 : 업로드 하는 엑셀 파일의 데이터를 합친 후 테이블에 노출
 * 시트별 구분 : 파일, 시트별 구분하여 테이블 노출
 * 엑셀 다운로드 : 데이터 프레임, 테이블 데이터 기준 다운로드

# dataframes.py
데이터 가공 메서드(Pandas 모듈)

# requiremodules.txt
설치기 필요한 모듈 모음 다운로드 후 터미널에 pip install -r requiremodules.txt 입력
